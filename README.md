# Photos App

Android photos management application.

## Features

- Create, delete, rename, and open albums
- Add and remove photos from albums
- View photos with slideshow navigation
- Add and delete tags (person and location only)
- Move photos between albums
- Search photos by tags with AND/OR logic and autocomplete

## Requirements

- Android Studio
- API 36
- Emulator: 1080 x 2400, 420 dpi
